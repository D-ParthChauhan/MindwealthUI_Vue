# GET /api/v1/macro/persistence

## Summary

Persistence / slow-grind signals (7-week grind, VIX suppression, etc.) and generic auto-discovered combo watch candidates.

## HTTP

- **Method:** `GET`
- **Path:** `/api/v1/macro/persistence`
- **operationId:** `get_persistence_signals`
- **Status:** implemented (v1.3.0)
- **Typical response:** 200, 404

## Maps to

`api/services/macro_service.get_persistence_signals()` → reads `runic_output.json`

## Response shape

```json
{
  "date": "2026-06-18",
  "persistence_signals": [
    {
      "signal_type": "VIX_SUPPRESSED",
      "weeks": 7,
      "description": "VIX below 18 for 7 consecutive weeks"
    }
  ],
  "generic_combo_watch": [
    {
      "vars": ["VXTS", "NFCI"],
      "status": "WATCH",
      "gate": "SIGNAL"
    }
  ],
  "source_freshness": {"last_audit": "2026-06-18"}
}
```

## Example

```bash
curl -s http://51.20.53.218:8506/api/v1/macro/persistence | jq '.persistence_signals'
```

## Notes

- `persistence_signals` are generated by `engine/persistence.py` — scans for multi-week patterns like sustained VIX suppression, grinding equity advance, etc.
- `generic_combo_watch` are auto-discovered variable combos from the 291 unnamed combinations that pass the ≥80% hit rate gate.
- Up to 10 generic combo watch candidates are surfaced (limited in the nightly payload).
